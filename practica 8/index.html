Arduino UNO Electrónica Robótica Bluetooth CNC Mecánica

- Resumen -

Esta práctica se enfocó en la implementación de una máquina CNC diseñada para realizar dibujos de forma automatizada. El sistema fue controlado mediante comunicación Bluetooth, lo que permitió enviar comandos desde una computadora al microcontrolador. A través de este control, se gestionó con precisión el movimiento de los motores paso a paso, permitiendo la creación de trazos sobre una superficie de dibujo y demostrando la capacidad del sistema para ejecutar patrones predefinidos de manera eficiente.

- Introducción -

En esta práctica se diseñó y fabricó una máquina CNC con el objetivo de realizar dibujos automatizados sobre una superficie plana. El sistema fue controlado mediante comunicación Bluetooth, lo que permitió enviar instrucciones desde una computadora hacia el microcontrolador encargado del movimiento de los motores paso a paso. Esta implementación demuestra cómo es posible replicar patrones predefinidos con exactitud, destacando el potencial de la automatización en aplicaciones gráficas.

Además, la experiencia permitió reforzar conocimientos sobre control de motores, diseño mecánico y programación de sistemas embebidos.

- Materiales -

Placa Arduino UNO.

Motores a paso 17 nema.

2 Cables para motores a paso.

Controlador.

2 Drivers.

Correa dentada.

Polea dentada.

Piezas de impresión 3D.

MDF.

Fuente de alimentación externa.

- Desarrollo -

El proceso comenzó con el diseño mecánico de la máquina CNC, tomando en cuenta las dimensiones necesarias para el área de trabajo y la disposición de los ejes de movimiento. Una vez finalizado el diseño, se procedió al corte de piezas estructurales en MDF, que conforman la base y el soporte de los componentes. Paralelamente, se imprimieron en 3D diversas piezas funcionales, como soportes para motores, acoples y guías para los ejes, optimizadas para su integración con los elementos mecánicos.

Posteriormente, se realizó el ensamble de los ejes de movimiento, integrando motores paso a paso, poleas y correas de distribución para garantizar un desplazamiento preciso y sincronizado en los planos X e Y. Se cuidó la alineación y fijación de cada componente para asegurar el correcto funcionamiento del sistema.

Una vez ensamblada la parte mecánica, se procedió al cableado de los motores, conectándolos a una placa Arduino, la cual estaba equipada con una shield que permitía la conexión de dos drivers para el control de los motores paso a paso. Finalmente, se estableció la comunicación entre la computadora y la placa Arduino mediante conexión por puerto serial. Desde la computadora, se ejecutó el programa que contenía las instrucciones de movimiento, permitiendo controlar de forma remota el trazo de dibujos mediante comandos precisos enviados al sistema.

- Diseño -

Para la base y el marco se utilizó MDF cortado a medida, lo cual proporcionó estabilidad y facilidad de montaje.

Se integraron también piezas impresas en 3D, diseñadas específicamente para alojar componentes clave del sistema, como los soportes para los motores paso a paso, las guías de deslizamiento y el mecanismo de sujeción del plumón. Estas piezas permitieron una unión precisa entre elementos y facilitaron el ensamble modular del equipo.

El sistema de movimiento está compuesto por dos ejes (X y Y), cada uno accionado por motores paso a paso conectados a poleas y correas de distribución, lo que garantiza un desplazamiento suave y coordinado. El eje Z, que sostiene el plumón, también está conformado por piezas impresas que permiten su ascenso y descenso, ajustando el contacto con la superficie de dibujo según sea necesario.

Construcción

Se ensamblaron los motores junto con las poleas dentadas. Posteriormente, se colocó la correa entre los ejes. El eje X es el que se mueve, mientras que el eje Y permanece estático. Para el control con Arduino, se utilizó una placa Arduino CNC Shield junto con dos drivers. Los cables de los motores se conectaron a los terminales correspondientes a los ejes X y Y en la placa, la cual fue montada directamente sobre el Arduino.

- Resultados -

El sistema funcionó correctamente, ya que logró pintar el cuadrado de manera precisa según lo esperado.

- Conclusiones -

Se cumplió exitosamente el objetivo de la práctica, logrando ensamblar y controlar el sistema CNC de manera correcta. La integración de los motores, poleas y la placa Arduino CNC Shield permitió que el equipo realizara movimientos precisos, como pintar el cuadrado con exactitud. Esto demuestra la efectividad del control mediante Arduino para automatizar procesos y garantizar un funcionamiento confiable.
