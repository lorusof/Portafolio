Resumen:

Hemos adquirido conocimientos fundamentales en la programación y control de dispositivos con Arduino. Iniciamos con el lenguaje básico para gestionar componentes electrónicos simples y progresamos hasta el control de elementos más complejos, como servo motores. Luego, aplicamos estos conocimientos en un proyecto más avanzado: el diseño y construcción de un brazo robótico controlado mediante potenciómetros. Finalmente, exploraremos la integración de Arduino con dispositivos móviles mediante el módulo HC-05 y una aplicación desarrollada en App Inventor, permitiendo la comunicación y control remoto desde un teléfono celular.
