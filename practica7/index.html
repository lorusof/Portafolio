Brazo Robotico con App Bluethooth

Rojas Robles Iván
Ingenieria Mecatronica
Fecha: Marzo 28, 2025
Arduino Electrónica App Installer

- Resumen -

El proyecto consiste en el desarrollo de una aplicación móvil utilizando App Inventor para controlar un brazo robótico a través de Bluetooth. La interfaz de la aplicación permite mover cuatro servomotores, que controlan el brazo robótico, dentro de un rango de 0 a 180 grados. Los comandos se envían de forma inalámbrica al Arduino Uno, que está conectado al módulo Bluetooth HC-05. A través de esta conexión, el usuario podrá controlar el movimiento del brazo robótico de manera remota desde su dispositivo móvil.

- Introducción -

App Inventor es una plataforma de desarrollo de aplicaciones móviles basada en bloques de programación que permite crear aplicaciones para dispositivos Android sin necesidad de escribir código en lenguajes complejos. Este proyecto tiene como objetivo integrar App Inventor con un Arduino Uno para controlar un brazo robótico utilizando Bluetooth. A través de esta solución, se busca combinar la simplicidad de App Inventor para crear la interfaz de usuario y la potencia de Arduino para controlar los servos del brazo robótico.

- Materiales -

App Inventor: Plataforma para diseñar y programar aplicaciones móviles visualmente mediante bloques de programación. Se utilizará para crear la interfaz de usuario y controlar la comunicación Bluetooth.

Dispositivo Android con Bluetooth: El dispositivo móvil se usará para ejecutar la aplicación creada en App Inventor y conectarse al Arduino mediante Bluetooth.

Arduino Uno: Microcontrolador que controla los servos del brazo robótico, recibiendo los datos enviados desde la aplicación.

4 Servomotores: Estos actuadores se utilizan para mover las articulaciones del brazo robótico. Cada servo controlará un segmento del brazo.

Módulo Bluetooth HC-05: Este módulo permite la comunicación inalámbrica entre el Arduino y el dispositivo móvil. Se conecta al Arduino y recibe los comandos enviados desde la aplicación móvil.

Conexión a Internet: Para acceder a App Inventor y descargar cualquier software adicional necesario para el desarrollo y la carga del código en el Arduino.

- Desarrollo -

1. Creación de la Interfaz de la Aplicación en App Inventor: Nuevo Proyecto en App Inventor: Se comienza creando un nuevo proyecto en App Inventor. Esta herramienta proporciona un entorno visual donde se pueden agregar componentes como botones, controles deslizantes (sliders) y conectores Bluetooth. Diseño de la Interfaz: Se agrega un componente de BluetoothClient para la conexión con el módulo HC-05. Luego, se crean cuatro controles deslizantes (sliders), cada uno representando un servo del brazo robótico. Cada slider permitirá al usuario ajustar el ángulo de los servos de 0 a 180 grados. Botones de Conexión y Desconexión: Se añaden dos botones: uno para conectar el dispositivo móvil al Bluetooth y otro para desconectarlo. Ejemplo de Diseño: Cuatro Sliders: Cada uno para controlar un servo, con el rango de valores de 0 a 180 grados. Botón "Conectar": Inicia la conexión Bluetooth con el módulo HC-05. Textos de Visualización: Mostrar en tiempo real el valor actual de cada slider, lo que permite al usuario ver la posición exacta de cada servo.

2. Configuración de los Bloques en App Inventor: Bloques de Inicialización: Se configuran bloques para inicializar el BluetoothClient y establecer la conexión con el módulo HC-05. Esto incluye la selección del dispositivo Bluetooth y la conexión a él. Bloques de Conexión Bluetooth: Para conectar el dispositivo móvil al módulo Bluetooth, se utilizan bloques que escanean los dispositivos Bluetooth disponibles y se conectan al HC-05. Envío de Datos: Se configuran bloques para enviar los valores de los controles deslizantes (0 a 180 grados) cada vez que el usuario mueva un slider. Estos valores serán transmitidos al Arduino para controlar la posición de los servos. Desconexión de Bluetooth: Se añaden bloques para desconectar el Bluetooth cuando el usuario lo desee.

3. Configuración del Código en Arduino: Lectura de Datos desde Bluetooth: El Arduino recibirá los datos enviados desde la aplicación a través del módulo Bluetooth HC-05. Estos datos representarán los valores de los ángulos de los servos. Control de Servos: El Arduino moverá los servos según los valores recibidos, ajustando sus posiciones entre 0 y 180 grados. Códigos de Servo: Usando la librería Servo.h, el Arduino podrá controlar los servos de forma precisa.

- Resultados -

La aplicación permitió controlar los servos del brazo robótico de forma remota utilizando los sliders en la interfaz. La conectividad Bluetooth funcionó correctamente, y los valores enviados desde la aplicación se recibieron y procesaron por el Arduino para mover los servos. Los servos respondieron de manera precisa a los valores de los controles deslizantes, permitiendo un control eficiente del brazo robótico.

- Conclusiones -

El proyecto demostró cómo se puede controlar un brazo robótico utilizando una aplicación desarrollada en App Inventor y Bluetooth. La integración de estos dos sistemas, junto con el Arduino, permitió crear una solución inalámbrica simple y efectiva para controlar dispositivos electrónicos como servomotores. Además, se resaltó la importancia de las conexiones inalámbricas en proyectos de robótica, permitiendo la interacción en tiempo real con el dispositivo.
